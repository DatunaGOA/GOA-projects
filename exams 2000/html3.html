HTML ფუნქციები - მაგალითი

ეს მაგალითი აჩვენებს სხვადასხვა HTML ელემენტს და მათი ფუნქციებს:

  • <!DOCTYPE> - დოკუმენტის ტიპის დეკლარაცია
  • <html> - HTML დოკუმენტის ძირითადი ელემენტი
  • <head> - დოკუმენტის metadata ელემენტები
  • <meta> - metadata (მაგ. კოდირება, viewport) აღწერა
  • <title> - დოკუმენტის სათაური
  • <body> - შიგთავსი, რომელიც ჩანს ვებგვერდზე
  • <h1> - ყველაზე მნიშვნელოვანი სათაური
  • <p> - პარაგრაფი
  • <ul> - უწყვეტი სიის დასაწყისი
  • <li> - სიის ელემენტი
  • <a> - ჰიპერლინკი
  • <img> - გამოსახულების ჩასმა
  • <input> - მომხმარებლის შეყვანისათვის ან ინტერაქციისათვის
